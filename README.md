O **Dispensa Tech** é um SaaS (Software as a Service) focado na organização doméstica. Ele permite o controle rigoroso de estoque de alimentos, sugere receitas inteligentes com base nos itens disponíveis e automatiza a criação de listas de compras.

---

## ✨ Funcionalidades

- **CRUD de Ingredientes:** Cadastro, edição e exclusão de itens com controle de unidade e categoria.
- **Modo Dark/Light:** Interface adaptável com persistência de preferência no navegador.
- **Chef Virtual:** Consulta à API internacional de gastronomia com tradução automática para português.
- **Lista de Compras Automática:** Identificação de itens esgotados e geração de relatório pronto para compartilhamento.
- **Filtro em Tempo Real:** Busca instantânea por nome de ingrediente via JavaScript.
- **Alertas Visuais:** Sistema de cores (badges) para estoque baixo ou zerado.

## 🛠️ Tecnologias Utilizadas

- **Backend:** Python 3.10+ e Framework Flask.
- **Banco de Dados:** SQLite (com SQLAlchemy para ORM).
- **Frontend:** HTML5, CSS3 (Variáveis e Flexbox) e Vanilla JavaScript.
- **Integrações:** API TheMealDB e Deep Translator.
